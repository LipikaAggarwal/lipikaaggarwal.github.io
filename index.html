LLM Memory Estimator

The LLM Memory Estimator is a web application that calculates GPU memory requirements for large language models (LLMs). Users can input the number of parameters and select quantization techniques, receiving immediate memory estimations. The tool features an intuitive interface, comprehensive documentation on memory calculations, and state-of-the-art memory optimization techniques, making it suitable for both novice and experienced users.

Bill Counter

The Bill Counter is a web application built on Node.js that streamlines the process of counting and managing bills. It features a secure login and signup page, allowing users to create personalized accounts. With an intuitive interface, users can easily input bill amounts and receive a total count, enhancing efficiency in bill management. This project exemplifies practical web development skills and user-focused design.

Bank Management Systemt

The Bank Management System is a Python application with a graphical user interface (GUI) built using Tkinter. It leverages MySQL for database management, allowing users to create and manage bank accounts efficiently. Key functionalities include account creation, balance inquiry, deposits, withdrawals, and transaction history, providing a comprehensive solution for banking operations. This project emphasizes user-friendly design and efficient data handling, making it suitable for both learning and practical applications in banking.
